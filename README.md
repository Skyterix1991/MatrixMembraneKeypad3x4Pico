# MatrixMembraneKeypad3x4GO
C/C++Matrix Membrane Keypad 3x4 implementation for Raspberry Pico RP2040.
Tested with RaspberryPi 4B 4GB.

Pin 1	-> GP 6 (Pin 9)

Pin 2	-> GP 7 (Pin 10)

Pin 3	-> GP 8 (Pin 11)

Pin 4	-> GP 9 (Pin 12)

Pin 5 -> GP 10 (Pin 14)

Pin 6	-> GP 11 (Pin 15)

Pin 7	-> GP 12 (Pin 16)
